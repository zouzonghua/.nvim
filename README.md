# nvim
My Neovim setttings

## 安装

### 一、手动安装

```sh
  cd ~
  git clone git@github.com:zouzonghua/nvim.git .config/nvim
```

打开 nvim，并执行 `:PlugInstall` 安装插件。

### 二、脚本安装

```
sh <(curl -L https://github.com/zouzonghua/nvim/raw/main/utils/install.sh)
```
